# 🔧 SENSOR CONTROL UNIT (SCU)

ระบบตั้งค่าและควบคุม Sensor สำหรับการใช้งานจริงในหน้างาน

---

## 📌 Overview

SCU ใช้สำหรับ:

* ตั้งค่าระยะตรวจจับของ Sensor
* ควบคุมการทำงานของระบบ Output
* ตรวจสอบสถานะการทำงานแบบ real-time

---

## 🚀 เริ่มต้นใช้งาน

1. เข้าไปที่:

```
Version1/
```

2. เปิด:

```
SCU1.exe
```

3. เสียบอุปกรณ์เข้าคอมพิวเตอร์

4. เลือก COM Port

5. ระบบจะเชื่อมต่ออัตโนมัติ

---

## ⚙️ วิธีใช้งาน

### 📥 อ่านค่า

* เมื่อเชื่อมต่อสำเร็จ
* ค่าปัจจุบันจะแสดงบนหน้าจอทันที

---

### 📤 ตั้งค่าใหม่

1. ใส่ค่า CH01 และ CH02
2. กด:

```
WRITE CONFIGURATION
```

---

## 💡 การแสดงสถานะ Sensor (สำคัญ)

ระบบมีไฟแสดงผลของ Sensor แต่ละตัว:

| สถานะไฟ     | ความหมาย            |
| ----------- | ------------------- |
| 🔵 / 🟢 ติด | Sensor “พบวัตถุ”    |
| ⚫ ดับ       | Sensor “ไม่พบวัตถุ” |

### วิธีดู:

* ถ้า Sensor ตัวไหน “ตรวจจับได้” → ไฟตัวนั้นจะติด
* ถ้า “ไม่มีอะไรอยู่ในระยะ” → ไฟจะดับ

👉 ใช้ดูได้ทันทีว่า Sensor ทำงานอยู่หรือไม่

---

## 📊 ความหมายของค่า

| ค่า  | ความหมาย                    |
| ---- | --------------------------- |
| CH01 | ระยะตรวจจับ Sensor ตัวที่ 1 |
| CH02 | ระยะตรวจจับ Sensor ตัวที่ 2 |

---

## 🔁 สถานะระบบ

| สถานะ     | ความหมาย           |
| --------- | ------------------ |
| CONNECTED | เชื่อมต่อสำเร็จ    |
| OFFLINE   | ยังไม่เชื่อมต่อ    |
| FAILED    | เชื่อมต่อไม่สำเร็จ |

---

## ❗ การแก้ปัญหาเบื้องต้น

### ไม่พบ COM

* กด SCAN
* เช็คสาย USB

---

### Sensor ไม่ติดไฟ

* ตรวจสอบว่ามีวัตถุอยู่ในระยะหรือไม่
* ตรวจสอบการเชื่อมต่ออุปกรณ์

---

### ค่าเป็น 0,0

* ยังไม่ connect
* หรืออุปกรณ์ไม่ตอบ
