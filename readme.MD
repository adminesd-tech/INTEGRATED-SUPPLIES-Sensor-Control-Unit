# 🔧 SENSOR CONTROL UNIT (SCU)

ระบบตั้งค่าและควบคุม Sensor สำหรับการใช้งานจริงในหน้างาน

---

## 📌 Overview

SCU ใช้สำหรับ:

* ตั้งค่าระยะตรวจจับของ Sensor
* ควบคุมการทำงานของระบบ Output
* ตรวจสอบสถานะการทำงานแบบ real-time

---

## 🚀 เริ่มต้นใช้งาน

1. เข้าไปที่โฟลเดอร์:

```
Version1/
```

2. เปิดโปรแกรม:

```
SCU1.exe
```

3. เสียบอุปกรณ์เข้าคอมพิวเตอร์

4. เลือก COM Port จากหน้าจอ

5. ระบบจะเชื่อมต่ออัตโนมัติ

---

## ⚙️ วิธีใช้งาน

### 📥 อ่านค่า

* เมื่อเชื่อมต่อสำเร็จ
* ค่าปัจจุบันจะถูกแสดงบนหน้าจออัตโนมัติ

---

### 📤 ตั้งค่าใหม่

1. กรอกค่าในช่อง:

   * CH01
   * CH02

2. กดปุ่ม:

```
WRITE CONFIGURATION
```

3. รอข้อความ:

```
CONNECTED / WRITTEN
```

---

## 📊 ความหมายของค่า

| ค่า  | ความหมาย                       |
| ---- | ------------------------------ |
| CH01 | ระยะตรวจจับของ Sensor ตัวที่ 1 |
| CH02 | ระยะตรวจจับของ Sensor ตัวที่ 2 |

---

## 🔁 การแสดงผลสถานะ

| สถานะ     | ความหมาย           |
| --------- | ------------------ |
| CONNECTED | เชื่อมต่อสำเร็จ    |
| OFFLINE   | ยังไม่เชื่อมต่อ    |
| FAILED    | เชื่อมต่อไม่สำเร็จ |

---

## ⚠️ ข้อควรระวัง

* ต้องเสียบอุปกรณ์ก่อนใช้งาน
* ห้ามถอดสายระหว่างกำลังใช้งาน
* ค่าที่ใส่ต้องเป็นตัวเลขเท่านั้น

---

## ❗ การแก้ปัญหาเบื้องต้น

### ไม่พบ COM Port

* ตรวจสอบสายเชื่อมต่อ
* กดปุ่ม SCAN

---

### เชื่อมต่อไม่ได้

* ตรวจสอบว่าเลือก COM ถูกต้อง
* ลองถอดแล้วเสียบใหม่

---

### ค่าเป็น 0,0

* ระบบยังไม่เชื่อมต่อ
* หรืออุปกรณ์ไม่ตอบสนอง
